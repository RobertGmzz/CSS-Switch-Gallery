Este párrafo muestra algunos elementos en línea que modifiqué ligeramente para que combinen con el resto de la plantilla. No todos los elementos están editados, algunos lucen bien por defecto, otros cumplen un rol semántico y no cambian la apariencia del texto. Verás mas ejemplos debajo.

14 de Febrero

El objetivo principal de CSS Switch Gallery es ofrecer una variedad de etilos predefinidos a los desarrolladores que necesitan una guía visual rápida de su web, antes de editar sus propios estilos CSS. O para aquellos que simplemente quieren un diseño rápido para su página estática, por ejemplo un portafolio personal. Tenga en cuenta que estos estilos no están hechos para un proyecto ya muy avanzado.

Para la demo, me basé en la referencia de elementos de MDN.

Listas, Ordenada y Desordenada

Características

  • Uso directo, no requiere instalación.
  • Puedes descargar la plantilla y modificarla si lo necesitas.
  • Incluye algunas clases opcionales para uso rápido.

Como usar

  1. Cambia los estilos del demo en tiempo real con los botones switch, y encuentra el que mas te guste.
  2. Copia el elemento <link> que se encuentra sobre el visualizador.
  3. Pégalo en el <head>de tu html.
Clases? Si, hay clases opcionales que incluí en las plantillas para utilizar de forma rápida si es preciso, la idea es la misma forma de implementar clases de tailwind. Verás ejemplos y su sintaxis a lo largo del demo.

Clases para botones

En muchas páginas web los botones tienen estilos diferentes dependiendo de la intención y relevancia. Aquí tienes clases para expandir el diseño de tus interfaces de usuario.

Utiliza las clases:

button-primary
button-secondary
button-tertiary
button-disabled
button-fit

No te dejes llevar por el nombre de las clases, no hay un boton que semanticamente sea mas importante que otro, solo les di ese nombre para diferenciarlos y facilitar su uso. Tú usalos según como creas conveniente.

Clases para diferenciar secciones de página

Finalmente, aqui tienes algunas clases para contenedores, es común ver que muchas webs utilizan distintos colores para determinada seccion de la pagina, o algo que necesitan destacar visualmente. Como en el caso de los botones, tu dales el uso que más te convenga.
